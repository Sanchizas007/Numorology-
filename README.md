# Aura Numerology PWA 🌌

Мистическое веб-приложение для расчета нумерологической карты судьбы.

## ✨ Особенности
- **PWA Ready**: Установка на главный экран смартфона.
- **Нумерологический движок**: Расчет Числа Жизненного Пути.
- **Генерация PDF**: Автоматическое создание отчета на стороне клиента.
- **Responsive Design**: Идеально выглядит на iOS и Android.

## 🛠 Технологии
- HTML5 / CSS3 (Custom Properties & Animations)
- JavaScript (Vanilla)
- [jsPDF](https://github.com/parallax/jsPDF) для формирования отчетов.

## 🚀 Деплой
1. Загрузите файлы в GitHub репозиторий.
2. Подключите репозиторий к **Render.com** (Static Site).
3. Готово!
